# PHY1001 力学

!!! note "课程描述"

    这是一门基于微积分的力学为主的基础工程物理课程，内容包括:力学中使用的物理单位和矢量概念，力和运动，功和动能，势能和能量守恒，动量和冲量，力矩和转动，力的平衡和弹性，流体，振动和机械波。
